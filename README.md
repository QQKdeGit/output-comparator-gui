# 代码对拍器

## 使用说明

首先把两份代码的`.cpp`文件以及能随机生成输入的`.cpp`文件放在根目录下

根据系统类型选择对应的操作

- Windows

  在根目录下打开 `Git Bash` ，输入指令

- Linux & macOS

  直接在终端里运行指令

指令如下

  ```shell
  bash comparator.sh input.cpp prog1.cpp prog2.cpp
  ```

> `input.cpp` 替换为能随机生成输入的文件名
>
> `prog1.cpp` 替换为第一份代码的文件名
>
> `prog2.cpp` 替换为第二份代码的文件名
